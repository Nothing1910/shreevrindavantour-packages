# Braj Map Section

The Braj Map Section is an interactive part of the website designed to help visitors understand the sacred geography of the Braj region in a simple and engaging way.

Instead of just reading about destinations, users can visually explore places like Mathura, Vrindavan, Govardhan, Barsana, and more on a map.

## How it Works

Each location is marked on the map.  
When a user clicks on a place, a short description appears explaining its importance.

When the user hovers over any location, the name of that place is shown instantly, making the experience smooth and intuitive.

## Tour Route Experience

Users can also select different tour types such as:

- 1 Day Tour  
- 2 Day Tour  
- Braj Yatra  

Once a tour is selected:

- Only the relevant places are highlighted  
- A path is shown between the locations  
- It becomes easier to understand how the journey flows  

This gives users a clear idea of what they will actually experience during the tour.

## Places Included

The map includes important spiritual destinations such as:

Mathura, Vrindavan, Govardhan, Barsana, Nandgaon, Gokul, Radha Kund, Shyam Kund, Kusum Sarovar, Seva Kunj, Nidhivan, and Vishram Ghat.

## Purpose

The goal of this section is to make the website more interactive and meaningful.

It helps visitors:

- Visualize the Braj region  
- Understand tour routes clearly  
- Feel more connected to the journey before booking  

This creates a better user experience and builds trust in the tour planning.
